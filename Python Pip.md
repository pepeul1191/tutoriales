# Python PIP

#### Instalación

Definir de manera global al sistema un usuario y correo.

    $ sudo apt-get -y install python-pip

### Instalación de Librerías

    $ sudo pip install pymongo
    $ sudo pip install Flask
    $ sudo pip install -U flask-cors
